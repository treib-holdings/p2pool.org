# p2pool.org

Source for [p2pool.org](https://p2pool.org) — a steward page for the domain that points visitors toward the projects working on decentralized Bitcoin mining in 2026.

## What this is

A single static HTML page plus one CSS file. No JavaScript, no tracking, no analytics, no fonts loaded from a CDN. The full payload is well under 20 KB.

## Corrections and suggestions

Open an [issue](https://github.com/treib-holdings/p2pool.org/issues) or a pull request. Both are welcome.

## Deployment

The site is hosted on GitHub Pages with `p2pool.org` as a custom domain. See [DEPLOY.md](DEPLOY.md) for the full setup, and a self-hosted DigitalOcean alternative.
